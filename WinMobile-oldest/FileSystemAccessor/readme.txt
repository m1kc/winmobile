Документация по использованию класса com.telecom.FileSystemAccessor

Moto_FS.zip
"Пустышка" для компиляции приложений для работы c файловой системой Motorola
telecom_MotoFS.zip
Класс com.telecom.FileSystemAccessor для работы c файловой системой Motorola

JSR75.zip
"Пустышка" для компиляции приложений для работы c файловой системой, спецификации JSR75
telecom_JSR75.zip
Класс com.telecom.FileSystemAccessor для работы c файловой системой, спецификации JSR75

При создании приложений для работы с ФС, необходимо скомпилировать приложение с нужной библиотекой:
telecom_MotoFS.zip или telecom_JSR75.zip. Исходный код самого приложения остается один и тот же!!!
